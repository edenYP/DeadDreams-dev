# DeadDreams

> *"This is just an idea i had and i really wanted to work on... so why not?"*

**DeadDreams** is a long-term solo game development project that I'm building while learning C++, Unreal Engine, and game development. (and while going through my college years)

This repository documents the **technical development** of the project, including gameplay systems, prototypes, design decisions, development logs, and progress updates.

The game's lore, worldbuilding, characters, and story are kept in a separate private repository because i do not want some greedy ahh dev or company stealing my work :D.

---

## End Goal

Create a memorable story-driven Action RPG where gameplay, atmosphere, and worldbuilding work together to create an unforgettable experience.
